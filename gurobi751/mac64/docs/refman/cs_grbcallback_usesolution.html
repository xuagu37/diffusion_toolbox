GRBCallback.UseSolution()

Once you have imported solution values using SetSolution, you can optionally call UseSolution to immediately use these values to try to compute a heuristic solution.

double UseSolution ( )

    Return value:

    The objective value for the solution obtained from your solution values (or GRB.INFINITY if no improved solution is found).
